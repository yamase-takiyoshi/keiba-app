競馬EV Pro GitHub Pages版

1. GitHubで keiba-app リポジトリを作る
2. index.html をアップロードする
3. Settings > Pages > Deploy from branch > main / root > Save
4. 公開URL: https://yamase-takiyoshi.github.io/keiba-app/
5. netkeiba-bridge.user.js を Tampermonkey / Userscripts / Stay に入れる
6. netkeibaの出馬表または結果ページで「🏆 競馬分析アプリへ送る」を押す

注意:
GitHub Pages単体ではnetkeibaを直接スクレイピングしません。
リアルデータ取得はUserscript側が担当します。
